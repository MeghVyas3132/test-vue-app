# Vue.js Test Application

A simple Vue.js 3 application built with Vite, demonstrating various Vue.js features and patterns.

## Features

- Vue 3 Composition API
- Component-based architecture
- Interactive dashboard with stats
- Responsive design
- Clean and modern UI
- Tab-based navigation

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/MeghVyas3132/test-vue-app.git
   cd test-vue-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `src/App.vue` - Main application component
- `src/components/` - Reusable Vue components
  - `Dashboard.vue` - Dashboard component with stats and items
- `src/assets/` - Static assets like images
- `public/` - Public static files
